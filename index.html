One-time Firebase Setup Required for Real-time Sync

This tracker uses Firebase Realtime Database (free) so every device sees changes instantly.
1. Go to console.firebase.google.com → Create project (free Spark plan) → Add Web App.
2. Copy your firebaseConfig into the <script type="module"> block at the top of this file.
3. In Firebase Console → Realtime Database → Create Database → choose Test mode (allows read/write).
4. Commit & push to GitHub. Done — all users share live data automatically.
